## f_eof

f_eof 函数测试文件是否结束。

```c
int f_eof (
  FIL* fp   /* [输入] 文件对象 */
);
```

### 参数

*   **fp**: 指向已打开的文件对象结构的指针。

### 返回值

如果读/写指针已到达文件末尾，`f_eof` 函数返回一个非零值；否则返回零。

### 描述

在此版本中，此函数是作为宏实现的。

```c
#define f_eof(fp) ((int)((fp)->fptr == (fp)->fsize))
```

### 快速信息

始终可用。
